Extension Overview

Click and Clear is a Chrome extension designed for efficiently deleting browsing data with a single click or keyboard shortcut.

1. Single Purpose Description

Purpose

One-click deletion of browsing data including history, cookies, cached files, downloads, and form data.

This extension allows users to quickly clear their browsing data through:

  • Direct click on the extension icon
  • Keyboard shortcut (Ctrl+Shift+D on Windows/Linux, Cmd+Shift+D on Mac)

The extension provides granular control over which data types to delete and supports multiple time ranges (all time, last hour, day, week, or month).

4. Host Permissions

Permission Declaration

"permissions": ["browsingData", "commands", "storage", "tabs"]

a) browsingData Permission CORE

Purpose

Access Chrome's browsing data deletion API

Justification

  • Core Functionality: The primary feature of the extension is to delete browsing data. This permission is essential and cannot be replaced.
  • User Request Only: Data is only deleted when the user explicitly clicks the delete button or uses the keyboard shortcut.
  • No Unauthorized Access: The extension does not read or transmit browsing data; it only triggers deletion.
  • Compliance: This permission is required by Chrome Web Store policy for data deletion features.

Data Types Handled

  • Browsing history
  • Cookies and site data
  • Cache
  • Downloads history
  • Form data
  • IndexedDB/Web SQL data
  • Service workers

Scope: Limited to local deletion only; no external data transmission

b) commands Permission CONVENIENCE

Purpose

Register and handle keyboard shortcuts

Justification

  • User Convenience: Keyboard shortcuts allow users to quickly delete data without opening the popup.
  • Accessibility: Provides alternative input method for users who prefer keyboard commands.
  • System-Level Only: This permission only registers shortcuts at the system level; it does not read keyboard input for any other purpose.
  • Transparency: Shortcuts are declared in the manifest and visible to the user.

Shortcuts Supported

  • Windows/Linux: Ctrl+Shift+D
  • Mac: Cmd+Shift+D

Scope: System-level keyboard events only; no content access

c) tabs Permission OPTIONAL

Purpose

Close open browser tabs when user requests

Usage

Closes all open tabs if "Open windows and tabs" option is selected

Justification

  • User-Controlled Feature: Users can optionally select "Open windows and tabs" for deletion.
  • No Mandatory Closing: Tabs are only closed when the user explicitly enables and triggers this option.
  • Privacy Benefit: Allows users to close browsing sessions that could reveal their activity.
  • Limited Scope: Only closes tabs; does not read tab content or history.
  • User Awareness: Clear UI checkbox informs users about this capability before deletion.

Scope: Only affects tabs when user explicitly triggers deletion; no background monitoring

d) storage Permission PREFERENCES

Purpose

Store user preferences for deletion settings

Data Stored

  • Selected data types to delete (binary flags)
  • Preferred time range selection
  • No personal or browsing information

Justification

  • Preference Preservation: Users expect their settings to be remembered between sessions.
  • Local Storage Only: Uses chrome.storage.local (device-only), not cloud sync.
  • Minimal Data: Only stores user preferences, not personal information.
  • User Control: Stored data can be cleared anytime through Chrome settings.
  • No External Transmission: All data remains on the user's device.

Scope: Local device storage only; no cloud synchronization or external access
